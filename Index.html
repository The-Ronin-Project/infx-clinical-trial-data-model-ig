Welcome to Ronin's Clinical Trials Data Model!


The Ronin Clinical Trials Data Model (CTDM) assembles a core set of data elements needed for clinical trials based on the Ronin Common Data Model (RCDM), which in turn is based on FHIR US Core and Version R4 standards.

The CTDM is not a true derivation of RCDM (a true derivation would be as strict or more strict than the original), but rather is a simplified version. The CTDM includes the modifications and extensions necessary to allow the electronic flow of data from electronic health record (EHR) systems to electronic data capture (EDC) systems used for clinical trials.

As of December 2023, the CTDM contains the single Observation profile that is needed for the Oracle Life Sciences Proof of Concept (POC) project. In the future, additional profiles will be added as needed.
